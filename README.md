# 2D Shooter Prototype

## Description
This is a 2D shooter prototype made in Unity. The game features:
- 3 enemy types (Chaser, Shooter, Tank)
- 2 weapons with different shooting patterns
- Pickups: Health and Score
- Progression system: 10 waves with increasing difficulty
- Win/Lose conditions

## Controls
- Move: Arrow keys / WASD
- Shoot: Space / Left mouse button

## Features
- Enemy AI with patrol, chasing, and shooting
- Player HP and score UI
- Wave management system
- Health pickups
- Two weapon types

## Build
- Unity 2025+ recommended
- Windows build included in the `Builds` folder
